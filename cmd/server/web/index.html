Что умеет сервис

  • 🔍
    WHOIS / RDAP

    Определяет регистратора доменов — через WHOIS или RDAP как фолбэк. Поддерживает все популярные зоны.

  • 🕵️
    Детект клоакинга

    Запрашивает каждый сайт тремя способами (без UA, Googlebot, Chrome+Referer) и сравнивает схожесть страниц. Низкий процент — сигнал клоакинга.

  • 🔗
    Canonical & Alternate

    Парсит теги <link rel="canonical"> и все alternate, показывает куда они ведут — та же страница, папка, поддомен или сторонний домен.

  • 📄
    Чекер вхождений

    Загрузите .txt со списком ключей — сервис посчитает их на странице отдельно для каждого типа запроса. Удобно для поиска микро-клоак по чувствительным словам.
